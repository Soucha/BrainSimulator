LTConditionalTarget

Name Reach cued target 
Implementing class LTConditionalTarget
Description Discriminate between possible targets and learn to go to the right one depending on a visual cue
Child equivalent Choose proper tool based on context
Child age 1.5 y
Prerequisites LTApproach
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary inputs
Spatial patterns yes   
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and two targets at random positions. Only one of the targets is associated with a reward. Which one it is depends on a condition (visual signal), e.g., a screen area displaying different colors
Training unit goal state The agent is close to the reward.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Level success criterion The pupil has not made a mistake in the last 20 training units larger than 120% of the optimal path length.
