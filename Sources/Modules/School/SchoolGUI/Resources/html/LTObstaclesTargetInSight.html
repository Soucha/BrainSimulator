LTObstacles

Name Reach occluded target
Implementing class LTObstaclesPOW
Description The agent learns to avoid obstacles while navigating to a reward. A simple verison of the task expects freestanding, solitary obstacles. A difficult version expects clusters that start to form a maze-like environment. But this taks should not be a maze.
Child equivalent Find way to bathroom
Child age 1.5 year
Prerequisites LTConditionalTarget, LTApproach
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary inputs
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and a target
Training unit goal state The agent should reach the target's coordinates
Reward There should be reward proportional to the number of steps taken in order to reach the target. Possibly a punishment for colliding with obstacles
Level success criterion The pupil has not made a mistake in the last 20 training units.
