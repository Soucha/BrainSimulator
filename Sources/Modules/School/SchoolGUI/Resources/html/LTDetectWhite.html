LTDetectWhite

Name Detect object presence        
Implementing class LTDetectWhite
Description Detect the presence of a simple pattern (a white square)
Child equivalent Detect person
Child age 1 week
Prerequisites none
Fully observable world yes
Input modalities Visual only
Actuators 2 (0,1). Both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying either an empty grid or a grid with a white square somewhere.
Training unit goal state The agent sets the actuator to 1 if the square is present.
Reward Reward for reaching goal state. Punishment for any other action.
Level success criterion The pupil has not made a mistake in the last 20 training units.
