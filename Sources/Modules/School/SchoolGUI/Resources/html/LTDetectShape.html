Name Categorize object shapes
Implementing class LTDetectShape
Description Categorize simple shapes
Child equivalent Recognize object
Child age 4 months
Prerequisites LTDetectBlackAndWhite
Fully observable world yes
Input modalities Visual only
Actuators 8 (1,2,3,4,5,6,7,8) - all binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an empty grid. At some position of the grid, there is either a square or a disc.
Training unit goal state The agent sets actuator A to 1 if the shape is a square; etc. Only one nonzero actuator is accepted.
Reward Reward for reaching goal state. Punishment for any other action.
Level success criterion The pupil has not made a mistake in the last 20 training units.
