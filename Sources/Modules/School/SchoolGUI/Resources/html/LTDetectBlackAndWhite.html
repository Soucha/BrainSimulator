LTDetectBlackAndWhite

Name Detect objects presence
Implementing class LTDetectBlackAndWhite
Description The agent learns to detect a black square on a random position from a white square on a random position. Initially, the squares are at pre-set positions and gradually they start to appear at different locations.
Child equivalent Differentiate father and mother
Child age 4 weeks
Prerequisites LTDetectWhite
Fully observable world yes
Input modalities Visual only
Actuators 2 (0,1). Both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an empty grid. At some position of the grid, there is either a black or a white square.
Training unit goal state The agent sets actuator A to 1 if the square is black. Otherwise (white square), he sets actuator B to 1.
Reward Reward for reaching goal state. Punishment for any other action.
Level success criterion The pupil has not made a mistake in the last 20 training units.
