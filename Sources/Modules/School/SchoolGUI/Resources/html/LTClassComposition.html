LTClassComposition

Name Categorize to arbitrary sets
Implementing class LTClassComposition
Description Perform action if an object shape from one arbitrary (fixed) set appears.
Child equivalent Return toys to a secretly selected boxes
Child age 1.6 y
Prerequisites LTDetectShape
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1). Both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an empty grid. At some position of the grid, there is an object. The shape of the object belongs to one of two possible classes. Different shapes from one class shoudn't be similar. Target is to determine, wheter outputs of different shape detectors can be processed afterwards. It's generalization based on reward.
Training unit goal state Agent set actuator A > 0 if shape belongs to first class, or B > 0 for second class, else if there is no shape set actuators to 0.
Reward Reward for reaching goal state. Punishment for any other action.
Level success criterion The pupil has not made a mistake in the last 20 training units.
