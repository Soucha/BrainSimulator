LTHiddenTarget

Name Reach hidden target 
Implementing class LTHiddenTarget
Description Discriminate between multiple possible targets and learn to pick the predefined one
Child equivalent Guess which toy I like.
Child age 1,8 year
Prerequisites Navigation to a target
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary inputs
Spatial patterns yes   
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and multiple possible targets at random positions. Only one of the targets is associated with a reward.
Training unit goal state The agent is close to the reward.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Level success criterion The pupil has not made a mistake in the last 20 training units larger than 120% of the optimal path length.
