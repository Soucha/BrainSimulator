LTTetrisFull

Name Unsupervised Tetris
Implementing class LTTetrisFull
Ability name Play Tetris
Description Agent is placed into Tetris World; there is no supervision and reinforcement signal. The agent is free to interact with the environment for a given amount of time.
Child equivalent -
Child age -
Prerequisites LTCompatibilityMatching, LTRotateAndMoveToFit
Fully observable world yes
Input modalities Visual
Actuators 2 (A,D), both binary
Spatial patterns yes  
Temporal patterns yes
Amount of supervision US
Distraction no
Training unit initial state
Training unit goal state The agent has erased a line from the stack of tetriminos
Stopping condition The stack of tetriminos reaches the top of the field
Reward None
Teacher Skill considered learned after a fixed number (by default 20) of consecutive successful training units. There is no progression of difficulty other than that inherent in the game.
