LTMultipleTargetsSequence

Name Reach multiple targets 
Implementing class LTMultipleTargetsSequence
Description The agent learns to navigate to multiple targets in a defined sequence. One target type is always the first target, another target type is always the second and so on.
Child equivalent Go to father, grandmother and back to me.
Child age 1.5 year
Prerequisites LTConditionalTarget
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary outputs
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL
Distraction no
Training unit initial state A training unit begins with a world displaying an agent at a default position and a number of targets
Training unit goal state The agent should reach the targets using the right sequence
Reward There should be reward proportional to the number of steps taken in order to reach the target. Possibly a punishment for colliding with the wrong target
Level success criterion The pupil has not collide with obstacles or passed the time-step limit in the last 20 training units.
